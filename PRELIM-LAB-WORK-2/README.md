# Sorting Algorithm Benchmark Tool

## Interface
Console-Based Program (CLI)

## Description
This Python program compares the performance of three sorting algorithms:
- Bubble Sort
- Insertion Sort
- Merge Sort

The user selects the algorithm and dataset size. The program measures execution time and verifies if the output is correctly sorted.

## How to Run
```bash
python main.py
```

## Requirements
- Python 3.x
(No external libraries required)
